# Gaceta - Backend

Servicio backend del sistema de **Gaceta**, encargado de la administración de publicaciones y almacenamiento de documentos PDF.

---

## Descripción

El backend del sistema **Gaceta** gestiona la lógica de negocio necesaria para registrar, administrar y publicar documentos oficiales en formato PDF. Además, expone los servicios y endpoints requeridos para que el frontend pueda consultar la información de las publicaciones.

---

## Funcionalidades principales

- Registro de publicaciones
- Carga de documentos PDF
- Almacenamiento de archivos
- Actualización de publicaciones
- Eliminación de publicaciones
- Consulta de publicaciones registradas
- Gestión de metadatos de documentos
- Exposición de API para integración con el frontend
